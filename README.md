## 简介
小红书的api都有加密，主要就是x-s。本项目是用python逆向小红书x-s，小红书会定期更新加密的js，本项目会持续更新，欢迎star。

## how to run

- python环境
  
  - execjs包
    
  - py_mini_racer 包
    
- java环境
  
- node js环境
  
- 你需要将自己需要的api、参数、cookie进行手动替换
